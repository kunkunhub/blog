## 访问方法  
### 开发时访问：  
首先确保安装了 python3  
接着安装依赖项:  
```
$ pip install -r requirements.txt  
```
然后需要设置SECRET_KEY:  
```
新建.env文件
写入
SECRET_KEY="<xxx>"
把<xxx>换成一个复杂的字符串
```
最后就能运行了:  
```
$ flask run
```
要注意，这个命令要在和这个README同级目录下执行
