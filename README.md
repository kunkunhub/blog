# 编程小站  
编程小站是由一些个人的编程教程、学习笔记、编程资源、编程工具整理得到的，由python的tornado框架编写。  


## 访问方法  
***
 __```pip install -i https://pypi.tuna.tsinghua.edu.cn/simple tornado```__  
 首先确保安装了python，然后在终端运行上面的命令来安装tornado，最后只要运行app.py就行。

## 目录结构  
***
> app.py       --网站服务端主程序  
> templates    --网页
